# Pairfon Mac Installer

Download the latest installer from the [**Releases**](https://github.com/saif-ullah-k/pairfon-installer/releases/latest) page.

1. Download **PairfonInstaller.zip**
2. Double-click to unzip → **Pairfon Installer.app**
3. First launch: **right-click the app → Open → Open** (one-time, unsigned build)
4. Follow the popups (Core URL, pairing token, Apple ID handle)

The agent is bundled inside the app — no source code or repo access required.
